# Fleet Memory Bibliography

References used by the fleet-memory paper set. These are additional to the main bibliography and focus on agent memory systems, distributed sync, and operational reliability.

## Agent-memory systems and frameworks

1. Mem0. "Universal memory layer for AI Agents." GitHub repository, https://github.com/mem0ai/mem0; product site, https://mem0.ai/.
2. D. Calca, J. Rees, and R. Morris. "Zep: A Temporal Knowledge Graph Architecture for Agent Memory." arXiv:2501.13956 [cs.CL], 2025.
3. C. Packer, S. Wooders, K. Lin, V. Fang, S. G. Patil, I. Stoica, and J. E. Gonzalez. "MemGPT: Towards LLMs as Operating Systems." arXiv:2310.08560 [cs.AI], 2023.
4. Q. Wu, G. Bansal, J. Zhang, Y. Wu, S. Zhang, E. Zhu, B. Li, L. Jiang, X. Zhang, and C. Wang. "AutoGen: Enabling Next-Gen LLM Applications via Multi-Agent Conversation." arXiv:2308.08155 [cs.AI], 2023.
5. CrewAI. "Memory — CrewAI." Documentation, https://docs.crewai.com/v1.15.1/en/concepts/memory.
6. O. Khattab, A. Singhvi, P. Maheshwari, Z. Zhang, K. Santhanam, S. Vardhamanan, S. Haq, A. Sharma, T. Joshi, H. Moazam, H. Miller, and M. Zaharia. "DSPy: Compiling Declarative Language Model Calls into Self-Improving Pipelines." arXiv:2310.03714 [cs.CL], 2023.
7. J. S. Park, J. C. O'Brien, C. J. Cai, M. R. Morris, P. Liang, and M. S. Bernstein. "Generative Agents: Interactive Simulacra of Human Behaviour." *ACM UIST*, 2023.

11. Redis. "Redis Pub/Sub." Redis Documentation, https://redis.io/docs/latest/develop/pubsub/.
